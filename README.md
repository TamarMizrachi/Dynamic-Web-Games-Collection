# Games & Puzzles Website - JavaScript Web App

## Overview

This project is a web application developed as part of a web development course.
It features a collection of interactive games and puzzles, allowing users to register, log in, and play different activities through a dynamic and user-friendly interface.

## Features

* **User Authentication**: Registration and login system with form validation
* **Local Storage Integration**: User data is stored and managed locally
* **Multiple Games**: Collection of games, puzzles, and interactive activities
* **Game Interface**: Each game includes instructions, controls, and gameplay elements
* **Score & Time Tracking**: Basic tracking of scores and/or time (depending on the game)
* **Dynamic UI**: Interactive elements using DOM manipulation and events
* **Responsive Design**: Basic adaptation for different screen sizes (if implemented)

## Technologies Used

* **Frontend**: HTML5, CSS3, JavaScript
* **Core Concepts**:

  * DOM Manipulation
  * Event Handling
  * Functions & Arrays
  * Timers & Dates
  * Local Storage
* **Styling**: CSS (Flexbox, animations, transitions)

## Project Structure

* Multiple HTML pages (landing page + game pages)
* Separate CSS files for styling
* Separate JavaScript files for logic and interactions
* Assets folder (images, media, etc.)

## Getting Started

1. Clone the repository
2. Open the project folder
3. Run the application by opening:

   ```
   index.html
   ```
4. Navigate through the website and play the games

## Notes

* This project is fully client-side
* Data is stored using Local Storage
* The focus is on functionality, interactivity, and DOM manipulation
* Some games are implemented more deeply than others, as required
